
# Assignment Test - PT. Xsis MItra Utama

This is a [Laravel app](https://laravel.com/) with version 8.


## Run Locally

Clone the project

```bash
  git clone https://github.com/xcodeme21/xsis.git
```

Go to the project directory

```bash
  cd xsis
```

Setting Database

```bash
  cp .env.example .env
  update DB Settings
```

Install package with Composer

```bash
  composer install
```

Start the server

```bash
  php artisan serve
```

Start Unit Test

```bash
  php artisan test
```
